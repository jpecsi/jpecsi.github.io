Projects

KegWatch

Link to GitHub Project

KegWatch is a sensor I built to keep track of how much beer is left in my kegerator. Instead of using weight, I track how long tap handles are open (using reed switches) and estimate how many oz were poured based on flow rate.

HopperHawk

Link to GitHub Project

HopperHawk is a sensor I built to keep track of how full the hopper is on my pellet smoker. I use an ultra sonic sensor to periodically measure how many pellets are left.
